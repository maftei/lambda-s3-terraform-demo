# Lambda + S3 + Terraform Demo

This project demonstrates a complete AWS Lambda setup managed with Terraform.

## What it does
- Creates an S3 bucket
- Uploads a file to S3
- Creates an IAM role with least privilege
- Creates a Lambda function
- Lambda reads the file from S3 and prints its content

## Tech stack
- AWS Lambda
- Amazon S3
- IAM
- Terraform
- Python
- LocalStack (for local development)
